# Banco de Questões — IBGP

> Questões da banca atual (IBGP) ou de concursos por ela aplicados.

## Onde encontrar

- Site oficial **IBGP**: `ibgpconcursos.com.br/concurso.jsp` → buscar concursos passados de carreiras jurídicas
- Site **MPMG**: `mpmg.mp.br/portal/menu/servicos/concursos/membros/`
- **QConcursos** (`qconcursos.com`) → filtrar banca = "IBGP"

## Como organizar aqui

Sugestão de estrutura:
- `LXI-MPMG-2024.md` — todas as questões de Processo Civil dessa prova
- `LX-MPMG-2018.md` — concurso anterior do MPMG
- `outros-IBGP-XXXX.md` — concursos não-MPMG aplicados pela banca

Para cada questão registrar (se quiser fora do QConcursos):
```
### Q<n> — [tema do edital]
**Enunciado:** ...
**Alternativas:** A) ... B) ... C) ... D) ... E) ...
**Gabarito:** X
**Fundamento:** CPC art. X / Súmula Y / etc.
**Você acertou? (data 1):** ✓ / ✗
**Você acertou? (data 2):** ✓ / ✗
```
