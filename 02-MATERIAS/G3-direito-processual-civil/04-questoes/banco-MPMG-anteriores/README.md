## Estratégia de estudo com prova MPMG

1. Pegar a prova **inteira** (não só Processo Civil) — entender o nível geral
2. Filtrar **só Processo Civil** para o estudo direcionado
3. **Resolver SEM consultar nada** primeiro (simulando prova)
4. Conferir gabarito, registrar erros em `../log-erros.md`
5. Para cada erro: investigar artigo da lei + jurisprudência → criar flashcard
