# Banco de Questões — Outros Ministérios Públicos

> Para complementar o banco MPMG. Concursos de outros estados costumam cair temas similares.

## MPs com perfil mais próximo (recomendados para complementar)

- **MPSP** — banca FCC tradicional, MP forte
- **MPRJ** — banca tradicional, vasta jurisprudência
- **MPRS** — banca tradicional, conteúdo equilibrado
- **MPPR** — banca FCC, MP organizado
- **MPRN** — bancas variadas, costuma cair direito coletivo
- **MPSC** — banca CESPE/CEBRASPE em alguns anos
- **MPDFT** — banca CESPE

## Bancas a observar

- **FCC** — literalidade, súmulas, jurisprudência
- **CESPE / CEBRASPE** — certo/errado (estilo diferente)
- **VUNESP** — ponderado
- **FGV** — exigente em jurisprudência

## Como organizar

Subpasta por concurso (criar conforme baixa):
- `MPSP-2023.md`
- `MPRJ-2022.md`
- `MPRS-2021.md`
- ...

## Estratégia

- **Não coletar tudo de uma vez.** Comece com 1 prova de cada MP a partir de 2020.
- Foco nas questões de Processo Civil — pular outras matérias.
- Se uma questão está "ultrapassada" (lei mudou, jurisprudência mudou), descartar.
