# Log de Erros — Processo Civil

> Registro das questões erradas. **Esse é o documento mais importante depois do checklist.** A maior parte do estudo eficiente vem de revisitar erros, não de ler conteúdo novo.

## Como usar

1. Errou uma questão? Anotar AQUI no mesmo dia.
2. Marcar para revisão em **7 dias** e **21 dias**.
3. Se errar de novo no D+7 ou D+21, virar **flashcard Anki imediato**.
4. Mensalmente, releia este arquivo inteiro — padrões aparecerão.

## Formato de cada entrada

```
### YYYY-MM-DD — Q<id da questão> | Banca | Tópico edital
- Enunciado resumido: ...
- Sua resposta: alternativa X
- Resposta correta: alternativa Y
- Por que errou: (confundiu X com Y / não sabia / chute)
- Aprendizado: (em uma frase)
- Revisão D+7: ✓ / ✗
- Revisão D+21: ✓ / ✗
- Virou flashcard: ☐ / ☑
```
